# NexIDE VR

NexIDE VR is a Quest-friendly browser IDE designed to run directly inside Meta Quest Browser, mobile browsers, and desktop browsers. It lets you build, edit, preview, and manage web projects without needing a traditional PC setup.

## Live Demo

https://tyl-droid.github.io/NexIDE-VR/

## Current Version

**NexIDE VR Sigma**

## Features

- Multi-project workspace
- File explorer
- Folder support
- File tabs
- HTML/CSS/JS live preview
- Console log capture
- Error capture
- Search and replace
- Project templates
- AI Builder prototype
- VR Mode
- Live overlay
- Live notifications
- Dev tools panel
- DOM inspector
- CSS inspector
- Storage viewer
- Performance stats
- Asset manager
- GitHub info panel
- Theme toggle
- Import/export project JSON
- PWA manifest
- Offline service worker support
- Quest Browser compatible UI

## Built For

- Meta Quest 3
- Meta Quest Browser
- Android phones
- Tablets
- Desktop browsers
- GitHub Pages
